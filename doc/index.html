README

Introduction

EspnScrape is a data scraper built in Ruby. Tabular data is returned in String arrays. Methods are also provided to convert data to Hash objects to allow dot notation access if preferred.

Note: Currently only for NBA

Quick Start

Installation

> gem install espnscrape

Examples

require 'espnscrape'

Access a Boxscore

bs = EspnScrape.boxscore(400828991) # Returns multidimensional array

Access a Boxscore as an array of objects

symbols = EspnScrape::FS_BOXSCORE
bs_objects = EspnScrape.from_array2d(symbols, bs) # Returns array of Hashes
